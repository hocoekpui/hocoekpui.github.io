Redis常见数据结构与常用命令

2022-02-03 5 min read

常见数据结构

string，list，set，zset，hash，bitmap，geo ...

字符串常用操作命令

命令 命令格式 描述
set set key value [EX seconds] [PX milliseconds] [NX|XX] 赋值
get get key 取值
getset getset key value 取值并赋值
setnx setnx key value 不存在则赋值
append append key value 追加
strlen strlen key 获取字符串长度
incr incr key 递增数字
incrby incrby key increment 增加指定的整数
decr decr key 递减数字
decrby decrby key decrement 减少指定的整数

列表常用操作命令

命令 命令格式 描述
lpush lpush value [value ...] 从左侧插入列表
lpop lpop key 从列表左侧取出
rpush rpush key value [value ...] 从右侧插入列表
rpop rpop key 从列表右侧取出
lpushx lpushx key value 将值插入到列表头部
rpushx rpushx key value 将值插入到列表尾部
blpop blpop key timeout 从列表左侧取出，当列表为空时阻塞，可以设置最大阻塞时 间，单位为秒
brpop brpop key timeout 从列表右侧取出，当列表为空时阻塞，可以设置最大阻塞时 间，单位为秒
llen llen key 获得列表中元素个数
lindex lindex key index 获得列表中指定下标的元素
lrange lrange key start end 返回列表中指定区间的元素
lrem lrem key count value 删除列表中与指定值相等的元素 当count > 0时， 从列表左边开始删除；当count < 0时，从列表右边开始删除;当count = 0时，删除所有与指定值相等的元素
lset lset key index value 将列表指定位置的元素设置成指定值
ltrim ltrim key start end 对列表进行修剪
rpoplpush rpoplpush src des 从 src 列表右侧弹出并插入到 des 列表左侧
brpoplpush brpoplpush src des 从 src 列表右侧弹出并插入到 des 列表左侧，会阻塞
linsert linsert key BEFORE/AFTER pivot value 将值插入到列表插入到指定值之前或之后

集合常用操作命令

命令名称 命令格式 描述
sadd sadd key member [member ...] 为集合添加新成员
srem sadd key member [member ...] 删除集合中指定成员
smembers smembers key 获得集合中所有元素
spop spop key [count] 返回集合中一个随机元素，并将该元素删除
srandmember srandmember key [count] 返回集合中一个随机元素，不会删除该元素
scard scrad key 获得集合中元素的数量
sismember sismember key member 判断元素是否在集合内
sinter sinter key [key ...] 求多集合的交集
sdiff sdiff key [key ...] 求多集合的差集
sunion sunion key [key ...] 求多集合的并集

有序集合常用操作命令

命令名称 命令格式 描述
zadd zadd key [NX|XX] [CH] [INCR] score member [score member ...] 为有序集合添加新成员
zrem zrem key member [member ...] 删除有序集合中指定成员
zcard zcard key 获得有序集合中的元素数量
zcount zcount key min max 返回集合中score值在 [min,max] 区间 的元素数量
zincrby zincrby key increment member 在集合的member分值上加increment
zscore zsorce key member 获得集合中member的分值
zrank zrank key member 获得集合中member的排名（按分值从 小到大）
zrevrank zrevrank key member 获得集合中member的排名（按分值从 大到小）
zrange zrange key start stop [withscores] 获得集合中指定区间成员，按分数递增 排序
zrevrange zrevrange key start stop [withscores] 获得集合中指定区间成员，按分数递减 排序

字典常用操作命令

命令名称 命令格式 描述
hset hset key field value 赋值
hmset hset key field value [field value ...] 批量赋值
hsetnx hsetnx key field value 赋值[不存在]
hexists hexists key field 查看值是否存在
hget hget key field 获取一个字段值
hmget hget key field [field ...] 获取多个字段值
hgetall hgetall key 获取所有字段值
hdel hdel key field [field ...] 删除指定字段
hincrby hincrby key field increment 指定字段自增
hlen hlen key 获得字段数量

位图常用操作命令

命令名称 命令格式 描述
setbit setbit key offset value 设置key在offset处的bit值
getbit getbit key offset 获得key在offset处的bit值
bitcount bitcount key [start end] 获得key的bit位为1的个数
bitpos bitpos key bit [start] [end] 返回第一个被设置为bit值的索引值
bitop bitop operation destkey key [key ...] 对多个key 进行逻辑运算后存入destkey 中

地理位置常用操作命令

命令名称 命令格式 描述
geoadd geoadd key longitude latitude member [longitude latitude member ...] 添加地理坐标
geohash geohash key member [member ...] 返回标准的地理位置哈希串
geopos geopos key member [member ...] 返回成员经纬度
geodist geodist key member1 member2 [unit] 计算成员间距离
georadiusbymember georadiusbymember key member radius m|km|ft|mi [withcoord] [withdist] [withhash] [count number] [asc|desc] 根据成员查找附近的成员
